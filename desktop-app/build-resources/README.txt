Place your application icon files here:

- icon.png   (256x256 or 512x512 PNG for Linux/macOS)
- icon.ico   (Windows icon, multi-size: 16x16, 32x32, 48x48, 256x256)

You can generate an .ico file from a PNG using online tools like:
- https://convertico.com
- https://icoconvert.com

If no icon files are present, electron-builder will use a default icon.
